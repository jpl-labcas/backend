# Docker

This folder contains containerization (`Dockerfile`) and orchestration (`docker-compose.yaml`) support for the Python-based LabCAS backend.

👉 **Note:** Do NOT use this. This is experimental only. The backend requires Solr in order to run and Solr is only avaiable bound to "localhost" on edrn-labcas or labcas-dev, and neither of these hosts are allowed to run Docker.
